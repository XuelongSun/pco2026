### Lab 3 — 8-bit ALU and Flags  
### 实验3 — 8位ALU与标志位

#### 1. 实验目标

构建一个可复用的8位 ALU 子电路。该 ALU 将作为最终CPU的数据计算核心。

完成本实验后，应能够：

1. 由1位基本部件组合出8位算术电路；
2. 使用补码实现加法和减法；
3. 集成简单逻辑运算；
4. 生成并验证ALU的标志位(程序状态字PSW)；

本实验对应理论课[第1章：计算](../../Chapter-1/readme.md)，尤其是 ALU 集成。

#### 2. 知识点
- 串行进位加法器。
- 补码减法：`A - B = A + (~B + 1)`。
- 运算选择器与多路选择器。
- 零标志 `Z`：结果为0。
- 进位/借位标志 `C`：加法表示进位；减法中采用何种约定必须在报告中说明。

### 3. 建议ALU接口

建议 ALU 引脚如下：

| 引脚 | 位宽 | 方向 | 含义 |
|---|---:|---|---|
| `A` | 8 | 输入 | 第一个操作数 |
| `B` | 8 | 输入 | 第二个操作数 |
| `OP` | 3 | 输入 | 运算选择 |
| `Y` | 8 | 输出 | ALU结果 |
| `Z` | 1 | 输出 | 零标志 |
| `C` | 1 | 输出 | 进位/借位标志 |
| `N` | 1 | 输出 | 可选符号标志 |

建议运算编码：

| OP | 运算 |
|---:|---|
| `000` | `Y = A + B` |
| `001` | `Y = A - B` |
| `010` | `Y = A + 1` |
| `011` | `Y = A - 1` |
| `100` | `Y = A AND B` |
| `101` | `Y = A OR B` |
| `110` | `Y = A XOR B` |
| `111` | `Y = NOT A` |

(可自行更改和调整~)

### 4. 实验任务

#### 任务1：8位加减法器

使用实验1的全加器构建8位串行进位加法器。

然后加入减法支持。

#### 任务2：逻辑运算

实现8位按位 `AND`、`OR`、`XOR` 和 `NOT A`。

#### 任务3：运算选择

使用数据选择器（multiplexor）或等价选择电路，根据 `OP` 选择最终 ALU 输出。

#### 任务4：标志位

基于3-8译码器(decoder)实现：

- 当 `Y` 所有位均为0时，`Z = 1`；
- `C` 按照你在报告中定义的加法/减法约定生成；
- 可选 `N = Y[7]`。
